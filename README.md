# Laravel plus

## Install
```Bash
composer require laravel-plus/laravel-plus
```

### install with github on composer.json

```JSON
{
    "require": {
        "laravel-plus/laravel-plus": "dev-main"
    },
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/byancode/laravel-plus"
        }
    ],
}
```
